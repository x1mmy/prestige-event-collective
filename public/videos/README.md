# Videos

Put all **videos** for the site here (e.g. Our Work carousel). Next.js serves files from `public/` at the site root.

## Suggested files

- **Our Work carousel:** `work-1.mp4`, `work-2.mp4`, … (or your naming)

## Usage in code

- **`<video>`:**  
  `src="/videos/work-1.mp4"`  
  Use `loop`, `muted`, `playsInline` for autoplay on hover (see `WorkCarousel.tsx`).
