# Favicons

Add your Prestige favicon files here with these **exact names**:

| Save your file as   | Use this image                    |
|---------------------|-----------------------------------|
| `favicon.png`       | Main favicon (default)            |
| `favicon-dark.png`  | Dark mode (e.g. black bars)        |
| `favicon-light.png` | Light mode (e.g. dark background) |
| `apple-touch-icon.png` | Apple touch icon (white on black) |

- **favicon.png** — Used when no light/dark preference; can be a copy of `favicon-dark.png` or `favicon-light.png` if you only have one.
- **favicon-dark.png** — Shown when the user’s system is in dark mode.
- **favicon-light.png** — Shown when the user’s system is in light mode.
- **apple-touch-icon.png** — Used when the site is added to an Apple device home screen (e.g. your “prestige-apple-touch-icon” image).

After saving the files here, refresh the site; the new icons will be used automatically.
